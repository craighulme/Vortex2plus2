# Introduction
**Vortex2+2** is a performance-focused and general-purpose web extension designed to enhance the visual and functional experience of the **Vortex** platform. Born from a simple Dark Mode project, it has evolved into a community-driven toolkit for power users.

(Vortex2+2 is a fork of [Vortex+](https://github.com/Klingri/Vortex.Plus/releases) made by Inuk with less bloat and big features released early!)
The name Vortex 2 + 2 was made by @Venkelbrood on github

# Installation
### Step-by-Step:
1. **Download:** Go to the [Releases](https://github.com/inuk84/Vortex-2-plus-2/releases) page and download the latest `Source code (zip)`.
2. **Extract:** Unzip the folder to a permanent location on your PC.
3. **Open Extensions Page:**
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
   - Vivaldi: `vivaldi://extensions`
   - Opera/OperaGX: `opera://extensions`
   - Arc: `arc://extensions`
   - Helium: `helium://extensions`
4. **Developer Mode:** Toggle the **Developer mode** switch in the top right.
5. **Load Unpacked:** Click **"Load unpacked"** and select the folder you just extracted.

# Known Issues & Stability
The Vortex+ Web Extension doesn't automatically update on it's own because it doesn't exist on the chrome web store due to the strict requirements by Google.
It only has dark mode and shaders for now, and it's in the very early stages of development so expect bugs and thing's to not work untill I make it stable enough.
- **Performance:** Shaders are currently experimental and may cause lag on lower-end systems.
- **UI Glitches:** Some text visibility issues may persist on specific platform pages.

**Report bugs via GitHub issues or @inuk1 on discord, or the Vortex+ discord server.**

> [!IMPORTANT]
> **Compatibility:** Currently, The Extension is not compatible at all in Firefox and Firefox-based web browsers, Chromium works tho!
